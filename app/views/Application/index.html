Getting Started

Welcome to Metro Manila's GTFS Editor.

The General Transit Feed Specification (GTFS) defines a common data format for public transportation routes. GTFS "feeds" allow public transit agencies to publish their transit data and developers to write applications that use that data in an standardized way. GTFS can be used to power trip planners, time table publishers, and a variety of applications.

How do I start?

Once you have some basic information about a route, click on New Route above to get started.

To view all current routes for your agency or find a route to edit, click on Explore/Search Routes above.

Have a Question?

Click on Help icons on next to any step to view instructions or tips.

Or select Contacts above to send a question or comment to an administrator. If you have any suggestions or issues please let us know!
